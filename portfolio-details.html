About the Project

The objective of this project is to make an automatic tuner for a guitar, which is out of tune. Each string on a guitar has a unique desired audio frequency, to match the requiredfrequency we used a microphone sound sensor and a servo motor with the help of an Arduino microcontroller. A microphone sensor is placed near the string and the servo motoris attached to the tuning peg of the guitar. The sensor reads the frequency and the servomotor rotates the tuning peg till it reaches the desired frequency.
